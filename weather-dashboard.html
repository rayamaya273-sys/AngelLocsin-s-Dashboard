How to Connect Your ESP32

  1. Set up your ESP32 to collect data from your sensors
  2. Choose a data storage method:
    • Google Sheets with Apps Script
    • Firebase Realtime Database
    • ThingSpeak IoT Platform
    • Local server or SD card storage
  3. Modify the JavaScript code in this dashboard to fetch data from your chosen storage
  4. Update the API endpoint in the code to match your data source
