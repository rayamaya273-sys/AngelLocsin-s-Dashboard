How to Use This Dashboard with GitHub Pages

  1. Create a GitHub account at github.com if you don't have one
  2. Create a new repository and name it "weather-station-dashboard"
  3. Copy this HTML code and create a file named "index.html" in your repository
  4. Go to your repository Settings, then Pages
  5. Select the main branch as the source and click Save
  6. Your dashboard will be available at: https://[your-username].github.io/weather-station-dashboard
  7. Modify the JavaScript code to fetch data from your own data source

Note: This demo shows simulated data. To connect real data from your ESP32, you'll need to set up a data storage solution like Google Sheets or Firebase and modify the JavaScript code accordingly.
